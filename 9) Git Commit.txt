* Using 'git commit -m "message"'

Commits the local change in the git repository.
Note: This does not update the remote repository as yet.

* Using 'git commit -m "message" --amend"

Amends the previous commit eg. adding changes withing the previous commit so that you
don't have to make a new version unnecessarily.

* Using 'git log'
Logs the current version commit and all commits done prior to it
Does not log the commits done after the current version
Note: Use 'q' to get out of terminal

* Using 'git log --all'
Logs all commits

* Using 'git log --all --graph'
Logs all commits and shows the branching (if present)
